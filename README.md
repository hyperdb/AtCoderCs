# AtCoder for `C#`

```bash
dotnet new console -o "SampleConsole"
cd SampleConsole
code .

dotnet run
```

## 古いスタイルでプロジェクトを作成する

- [.NET 6+ での C# コンソール アプリ テンプレートの変更 - .NET | Microsoft Learn](https://learn.microsoft.com/ja-jp/dotnet/core/tutorials/top-level-templates)

```bash
dotnet new console --use-program-main -o "SampleConsole"
cd SampleConsole
code .

dotnet run
```
